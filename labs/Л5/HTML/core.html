Модуль core

Модуль core содержит основную логику приложения, включая текстовый редактор и проверку орфографии.

Классы:

  • TextEditor - Класс для работы с текстом, включая добавление, очистку и сохранение текста.
  • SpellCheck - Класс для проверки орфографии на русском и английском языках.
