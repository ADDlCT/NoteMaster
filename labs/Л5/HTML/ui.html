Модуль ui

Модуль ui отвечает за пользовательский интерфейс приложения, включая управление темами оформления и главное окно.

Классы:

  • Theme - Класс для управления темами оформления приложения.
  • MainWindow - Класс для создания главного окна приложения, включая текстовый редактор и меню.
